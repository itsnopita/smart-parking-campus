# Smart Parking Campus 🚗

Project ini adalah sistem parkir kampus berbasis scan QR deangan Laravel.

## 👤 Role User
- Admin
- Petugas
- Member



## ⚙️ Cara Install

1. Clone repo:
git clone https://github.com/itsnopita/smart-parking-campus.git

2. Masuk folder:
cd smart-parking-campus

3. Install:
composer install
npm install

4. Copy env:
cp .env.example .env

5. Setting database:
DB_DATABASE=smart_parking_unbaja
DB_USERNAME=root
DB_PASSWORD=



## 🗄️ Import Database

Import file:
database/smart_parking_unbaja.sql
